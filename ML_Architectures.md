# Архитектуры моделей машинного обучения для определения параметров пленки

Данный документ описывает стратегии реализации ML-моделей для предсказания динамической вязкости $\mu$ и средней толщины пленки $h_{final}$ на основе временных рядов толщины $h(t)$.

## Общие входные данные
Для всех вариантов используются следующие данные:
- **Временные ряды**: $h_i(t)$ для каждого датчика $i \in \{1, \dots, k\}$.
- **Координаты**: Позиции датчиков $x_i$ вдоль стенки.
- **Известные константы**: Плотность жидкости $\rho$ и коэффициент поверхностного натяжения $\sigma$.

---

## Вариант 1: Независимый бустинг (Baseline)
Использует CatBoost для предсказания по отдельным датчикам с последующим усреднением.

## Вариант 2: CNN-MLP с конкатенацией признаков
Использует 1D-CNN для извлечения признаков из сигналов датчиков с последующим объединением через Mean Pooling.

---

## Вариант 3: Иерархический механизм внимания (SOTA)
Наиболее продвинутый подход, использующий механизмы обработки временных рядов и пространственных зависимостей.

### Логика работы:
1. **Temporal Encoder (1D-CNN)**:
   - Каждый временной ряд $h_i(t)$ независимо обрабатывается сверточной сетью (1D-CNN) для выделения стабильных локальных признаков формы волны.
   - **Результат**: Вектор признаков $v_i$ для каждого датчика.
2. **Spatial Encoder (Transformer)**:
   - Векторы $\{v_1, \dots, v_k\}$ вместе с их координатами $\{x_1, \dots, x_k\}$ подаются в Transformer Encoder.
   - Механизм внимания позволяет модели анализировать взаимосвязи (корреляции, фазовые сдвиги) между датчиками.
   - **Результат**: Глобальный вектор состояния системы $V_{global}$.
3. **Финальный предиктор**:
   - $[V_{global}, \rho, \sigma] \rightarrow \text{MLP} \rightarrow \log_{10}(\mu)$.

**Плюсы**: Инвариантность к количеству датчиков, высокая точность, учет физической структуры течения.

Подробный анализ и результаты доступны в файле `ML_Variant3_Analysis.md`.

## Вариант 4: Обогащенный бустинг (Enriched Boosting)
Подход с глубоким инжинирингом физически значимых признаков (FFT, динамика, корреляции) и использованием градиентного бустинга.
