# Анализ реализации Модели Варианта 4: Обогащенный бустинг (Enriched Boosting)

Данный документ описывает результаты итеративного улучшения базовой модели за счет глубокого инжиниринга признаков (Feature Engineering). Вариант 4 представляет собой развитие Варианта 1, направленное на извлечение физически значимых характеристик сигнала.

## 1. Архитектура модели

В основе Варианта 4 лежит подход независимого бустинга (CatBoost), но с существенно расширенным вектором входных признаков.

- **Метод**: Каждый датчик в примере рассматривается как отдельный обучающий объект.
- **Агрегация**: Финальный результат для примера определяется как среднее арифметическое предсказаний всех активных датчиков.
- **Входные данные**: Обогащенный вектор признаков, состоящий из пяти групп:
    1. **Глобальные константы**: $\rho, \sigma$.
    2. **Нормализованный ряд**: $\frac{h(t) - h_{mean}}{h_{mean}}$ (100 точек).
    3. **Частотные признаки (FFT)**: Доминирующая частота, спектральная энергия, пиковая амплитуда.
    4. **Статистика временного ряда**: Амплитуда, дисперсия, стандартное отклонение.
    5. **Динамические признаки**: Средняя и максимальная скорость изменения $\Delta h$, дисперсия производной.
    6. **Пространственные связи**: Максимальная кросс-корреляция с соседним датчиком, временной сдвиг (lag), разность стандартных отклонений.

## 2. Итеративный процесс улучшения

Повышение точности осуществлялось поэтапно, что позволило оценить вклад каждой группы признаков.

| Этап | Добавленные признаки | $R^2$ (Log-Viscosity) | Прирост |
| :--- | :--- | :---: | :---: |
| **Baseline** | Сырые данные (Вариант 1) | $0.6570$ | - |
| **Этап 1** | FFT + Базовая статистика | $0.6688$ | $\uparrow 0.0118$ |
| **Этап 2** | Динамика (Производные $\Delta h$) | $0.6767$ | $\uparrow 0.0079$ |
| **Этап 3** | Пространственный контекст (Корреляции) | **$0.6790$** | $\uparrow 0.0023$ |

## 4. Итоговые выводы

1. **Приоритет признаков над архитектурой**: Для текущего объема данных глубокий инжиниринг признаков (Feature Engineering) оказался значительно эффективнее, чем использование сложных нейросетевых архитектур. 
2. **Физическая интерпретируемость**: Рост точности при добавлении FFT и пространственных корреляций подтверждает, что вязкость $\mu$ кодируется в частоте волн и скорости их перемещения между датчиками.
3. **Эффективность бустинга**: CatBoost продемонстрировал наилучшую устойчивость и обобщающую способность на табличных данных, созданных из физических дескрипторов.
4. **Граница точности**: Наблюдается постепенное замедление темпов роста $R^2$, что указывает на приближение к теоретическому пределу точности для данной конфигурации датчиков и объема выборки.

**Итог**: Вариант 4 является оптимальным решением для данной задачи, обеспечивая наивысшую точность предсказания динамической вязкости.
