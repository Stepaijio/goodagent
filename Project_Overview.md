# Обзор проекта: Динамика течения жидких пленок

## Тема
"Алгоритмы обработки информации о динамике изменений характеристик течения жидкостей в пленочных аппаратах"

## Цель
Разработать модель машинного обучения, способную определять параметры тонкой жидкой пленки, стекающей вертикально вниз по стенке под действием силы тяжести.

## Техническая концепция
- **Сценарий**: Жидкая пленка стекает вертикально вниз по стенке.
- **Сбор данных**: В разных точках вдоль стенки установлены датчики. Каждый датчик измеряет толщину пленки, формируя временной ряд.
- **Задача**: На основе расположения датчиков, временных рядов толщины и набора известных параметров модель должна предсказать неизвестные параметры пленки.
- **Текущий статус**: Разработка и обучение моделей машинного обучения для определения параметров пленки.

## Дорожная карта реализации
1. **Разработка физической модели (Текущий этап)**: Реализация математической/физической модели пленки для генерации синтетических данных. Это главный приоритет. Подробности разработки: `Physical_Model.md`.
2. **Создание датасета**: Использование физической модели для создания разнообразного набора сценариев течения.
3. **Обучение модели**: Обучение и валидация ML-модели для точного определения параметров по временным рядам датчиков.
