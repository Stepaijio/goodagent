# Обзор проекта: Динамика течения жидких пленок

## Тема
"Алгоритмы обработки информации о динамике изменений характеристик течения жидкостей в пленочных аппаратах"

## Цель
Разработать алгоритмы анализа данных на основе машинного обучения для оперативного мониторинга параметров тонкой жидкой пленки в пленочных аппаратах. Основной акцент сделан на использовании цифрового двойника (Digital Twin) физического процесса для генерации эталонного датасета и последующей валидации методов идентификации параметров.

## Техническая концепция
- **Сценарий**: Жидкая пленка стекает вертикально вниз по стенке.
- **Сбор данных**: В разных точках вдоль стенки установлены датчики. Каждый датчик измеряет толщину пленки, формируя временной ряд.
- **Задача**: На основе расположения датчиков, временных рядов толщины и набора известных параметров модель должна предсказать неизвестные параметры пленки.
- **Текущий статус**: Завершена разработка и валидация ML-моделей. Определена наилучшая архитектура (XGBoost на базе обогащенных признаков) с точностью $R^2 = 0.6875$.

## Дорожная карта реализации
1. **Создание цифрового двойника процесса (Симулятор)**: Реализация математической модели пленки для генерации контролируемого синтетического датасета. Это необходимо для обучения ML-модели в условиях отсутствия экспериментальных данных. Подробности разработки: `Physical_Model.md`.
2. **Формирование эталонного датасета**: Использование симулятора для создания разнообразного набора сценариев течения с точно известными параметрами.
3. **Разработка алгоритмов анализа данных**: Обучение и валидация ML-моделей для определения параметров по временным рядам датчиков.
