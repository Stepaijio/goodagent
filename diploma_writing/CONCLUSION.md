# ЗАКЛЮЧЕНИЕ

В данной работе реализован комплексный подход к решению задачи идентификации параметров тонкой жидкой пленки по данным локальных датчиков толщины. Поставленная цель по разработке модели машинного обучения для определения динамической вязкости и средней толщины пленки достигнута в полном объеме. В ходе исследования решены все основные задачи, что позволило создать работающий программный комплекс для анализа гидродинамических процессов в пленочных аппаратах.

Разработанная физическая модель на основе теории тонких слоев обеспечила достоверную генерацию синтетических данных. Внедрение гибридной численной схемы позволило преодолеть проблему неустойчивости расчетов при высоких числах Рейнольдса. Созданный автоматизированный генератор обеспечил формирование репрезентативного датасета с применением логарифмически-равномерного сэмплирования параметров. Это гарантировало охват широкого диапазона физических режимов течения и создало надежную базу для обучения алгоритмов машинного обучения.

Сравнительный анализ пяти архитектур нейросетевых моделей и алгоритмов бустинга выявил доминирующее влияние качества входных признаков над сложностью архитектуры. Установлено, что глубокий инжиниринг физических дескрипторов, включая спектральный анализ и расчет пространственных корреляций, обеспечивает более высокую точность, чем автоматическое извлечение признаков глубокими сетями. Наилучшие результаты продемонстрировала модель XGBoost с коэффициентом детерминации $R^2 = 0.6875$. Данный результат подтверждает эффективность сочетания физических знаний о процессе с методами градиентного бустинга.

Практическая значимость работы заключается в возможности неинвазивного мониторинга свойств жидкостей в реальном времени. Разработанный подход позволяет определять динамическую вязкость без прямого вмешательства в технологический процесс. Это открывает перспективы для автоматизации контроля качества в химической и нефтеперерабатывающей промышленности. Полученные результаты могут быть использованы для дальнейшего совершенствования систем управления пленочными аппаратами и оптимизации режимов тепломассообмена.
